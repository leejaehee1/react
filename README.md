### Before start
```
$ node -v
$ npm -v
$ npx -v
```


### installation
```
$ npm i axios
$ npm install react-router-dom
```



### Get start

```
$ npx create-react-app fd_react .
$ npm i
$ npm start
```





### Branch

- master : 배포환경(금요일 update)
- develop : 개발환경  (by master)
- feature/router : 작업중   (by develop)
