### Before start
```
$ node -v
$ npm -v
$ npx -v
```


### installation
```
// npm i 로 한번에 설치가능 (package.json 참고)
$ npm i axios
$ npm install react-router-dom
$ npm install xlsx
$ npm install material-table @material-ui/core --save
$ npm install @material-ui/icons
// $ npm install @material-ui/data-grid
```



### Get start

```
$ npx create-react-app fd_react .
$ npm i
$ npm start
```
